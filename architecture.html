Security

🔒 Supabase Auth

Email/password with bcrypt. JWT sessions. No passwords in app.

🛡 Row Level Security

Users can only access their own data. Enforced at DB level.

🛡 XSS Prevention

All inputs sanitized via escapeHTML() before rendering.

✅ 16 Constraints

CHECK constraints on all fields. Range + enum validation.

📝 Soft Delete

Records never permanently deleted. Audit trail on all actions.

🔐 Publishable Key Only

No secret keys exposed in frontend code.
